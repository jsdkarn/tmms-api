## Description

API สำหรับเก็บบันทึกการเบิก/ยืมวัสดุอุปกรณ์

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Tools

- NestJS
- mongoose
